CryptoSMT

An easy to use tool for cryptanalysis of symmetric primitives based on SMT/SAT solvers.

Download this project as a .zip file Download this project as a tar.gz file

This page gives an overview of how to install CryptoSMT and how to use it.

Installing

CryptoSMT invokes two other tools and needs to know the paths to the executables:

  • STP: https://github.com/stp/stp
  • CryptoMiniSat: https://github.com/msoos/cryptominisat

You can clone the repositories and build STP/CryptoMiniSat.

Currently no install script is used and you have to add the paths to the cryptosmt.py

Python Libraries

If you need to install YAML:

$ pip install pyyaml

or

$ sudo apt-get install python-yaml

How to use?

You can get a list of all the parameters 
$ python cryptosmt.py --help

A short overview of the parameters:

Parameter Description
--cipher name Specifies the cipher to use.
--rounds x The number of rounds used of the cipher
--wordsize x The size of the statewords in bits.
--mode x Determines the mode of operation. See table below for a description.
--sweight x The weight of the differential characteristic for the search to start.
--iterative Only search for iterative characteristics.
The tool currently supports the following modes of operation:
Mode Description
0 Searches for the minimal weight characteristic.
1 Searches for the minimal weight characteristic for an increasing number of rounds.
2 Outputs all characteristics for the given parameters.
4 Determines the probability of the differential for the given parameters.

Input Files

A second way to use the tool is by using input files in the YAML format. 
$ python cryptosmt.py --inputfile examples/*.yaml

A sample input file might look like this

---
cipher: speck
sweight: 10
rounds: 8
wordsize: 16
mode: 0
...

It is also possible to set variables to a fixed values. This is especially useful when trying to determine the probability of a differential.

fixedVariables:
- x0: "0x200020"
- y0: "0x080088"
- x15: "0x080888"
- y15: "0x000200"

Parameters from the inputfiles can be overwritten by using the commandline parameters.
