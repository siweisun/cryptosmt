CryptoSMT

An easy to use tool for cryptanalysis of symmetric primitives based on SMT/SAT solvers.

Download this project as a .zip file Download this project as a tar.gz file

This page gives an overview of how to install CryptoSMT and how to use it.

Installing

CryptoSMT invokes two other tools and needs to know the paths to the executables:

  • STP: https://github.com/stp/stp
  • CryptoMiniSat: https://github.com/msoos/cryptominisat

Currently no install script is used and you can simple add the paths by adding them to the main.py

Python Libraries

If you need to install YAML:

$ pip install pyyaml

or

$ sudo apt-get install python-yaml
